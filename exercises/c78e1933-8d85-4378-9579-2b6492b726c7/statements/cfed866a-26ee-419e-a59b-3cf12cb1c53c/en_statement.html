Faulty display

Problem

In a bingo room that draws numbers from 0 to 80, a problem arose that prevents the table where the drawn numbers are shown from working. O room manager solved the problem using a traffic light that pulled out of the nearest intersection. Thus, for each number drawn the traffic light comes on four times, representing the digits of the number in the base 3, starting with the least significant and ending with the most significant. The green color represents the number 0, the orange color represents the number 1 and the red color the number 2.

Task

Write a program that gives an integer greater than or equal to 0 and less than or equal to 80 represents the sequence of lights on at the traffic light.

Input

An integer greater than or equal to 0 and less than or equal to 80.

Output

The program should display as a result 4 lines with the words green , orange or red of the least digit significant to the most significant digit.

Example 1: input 


15

Example 1: output 


green
Red
Orange
green

Example 2: input 


80

Example 2: output 


Red
Red
Red
Red

Example 3: input 


30

Example 3: output 


green
Orange
green
Orange
