Mostrador avariado

Problema

Numa sala de bingo que sorteia números do 0 ao 80 surgiu um problema que impede que o quadro onde são mostrados os números sorteados funcione. O gerente da sala resolveu o problema recorrendo a um semáforo de trânsito que arrancou do cruzamento mais próximo. Assim, para cada número sorteado o semáforo acende quatro vezes, representando os dígitos do número na base 3, começando com o menos significativo e acabando no mais significativo. A cor verde representa o número 0, a cor laranja representa o número 1 e a cor vermelha o número 2.

Tarefa

Escrever um programa que dado um inteiro maior ou igual a 0 e menor ou igual a 80 representa a sequência de luzes acesas do semáforo.

Input

Um inteiro maior ou igual a 0 e menor ou igual a 80.

Output

O programa deverá apresentar como resultado 4 linhas com as palavras verde, laranja ou vermelho do dígito menos significativo para o dígito mais significativo.

Exemplo 1: input 


15

Exemplo 1: output 


verde
vermelho
laranja
verde

Exemplo 2: input 


80

Exemplo 2: output 


vermelho
vermelho
vermelho
vermelho

Exemplo 3: input 


30

Exemplo 3: output 


verde
laranja
verde
laranja
