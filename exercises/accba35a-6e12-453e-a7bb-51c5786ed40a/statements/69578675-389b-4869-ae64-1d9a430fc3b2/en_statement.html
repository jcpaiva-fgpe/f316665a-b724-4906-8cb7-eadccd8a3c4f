Triangles

Three values ​​define the measurements of the sides of a triangle if and only if are positive and any of the values ​​is less than the sum of the two others.

Task

Write a program that, given K suits integers, check whether or not each suit defines the measurements of the sides of a triangle. If defined, you should check if it is an equilateral triangle (three equal sides). Each A B C suit is given on one line and each pair of integers in the suit is separated by a space. The first line has the number of suits. As appropriate, the program writes `` Yes, equilatero. '', `` Yes. '', or `` Of course no! '', followed by a newline.

Example

Input 


4
-1 4 5
7 7 7
4 3 6
8 2 1

Output 


Of course not!
Yes, equilatero.
Yes.
Of course not!
