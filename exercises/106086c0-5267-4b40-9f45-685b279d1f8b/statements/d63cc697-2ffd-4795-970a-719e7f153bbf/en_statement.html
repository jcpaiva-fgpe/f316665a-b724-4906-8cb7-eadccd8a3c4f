Good luck

Problem

The sportsmanship of the Portuguese is well known. So, intending to accompany the national team to Germany, some Portuguese have learned to say `` good luck '' in several languages: 
        GOOD LUCK

        BUENA SUERTE

        BONNE CHANCE

        BUONA FORTUNA

        VIEL GLUECK

Assignment

Write a program that reads two integers n and l and write n times good luck in the l language (in which l represents the order number in the previous list).

Input

In the first line there is the positive integer . On the line next has the value of l (where ).

Output

There will be n lines, with exactly the same words, each line ending by line change.

Example 1

Input 


4
two

Output 


BUENA SUERTE
BUENA SUERTE
BUENA SUERTE
BUENA SUERTE

Example 2

Input 


1
4

Output 


BUONA FORTUNA
