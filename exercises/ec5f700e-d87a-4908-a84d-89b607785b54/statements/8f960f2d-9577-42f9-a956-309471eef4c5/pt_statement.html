Problema A
Números triangulares

Problema

Um número inteiro positivo diz-se triangular se para algum . Por exemplo, é triangular porque , mas não é triangular.

Tarefa

Escrever um programa que dado um inteiro positivo diga se é triangular ou não.

Input

O input consiste num inteiro positivo .

Output

O seu programa deve escrever SIM ou NAO conforme o inteiro de input é ou não um número triangular.

Exemplo de Input 

28

Exemplo de Output 

SIM
