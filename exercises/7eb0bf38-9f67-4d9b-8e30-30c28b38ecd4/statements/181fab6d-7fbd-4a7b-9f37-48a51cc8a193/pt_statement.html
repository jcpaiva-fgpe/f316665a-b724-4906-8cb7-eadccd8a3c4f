Fogo Cruzado


Problema

Considere um tabuleiro com L linhas e C colunas. Na primeira linha está colocada numa das colunas uma bola branca que se desloca verticalmente ao longo dessa coluna e uma posição de cada vez. Nas restantes linhas (em qualquer coluna), poderão estar colocadas bolas pretas (no máximo uma por linha), representando projectéis. As bolas pretas movem-se apenas horizontalmente, uma posição de cada vez. As bolas pretas movimentam-se a partir de uma coluna inicial dada e segundo um sentido inicial também dado. Quando uma bola preta atinge o limite do tabuleiro esta inverte o sentido do seu movimento.


Tarefa

Escrever um programa que, dadas as dimensões do tabuleiro inicial L*C, a coluna onde se encontra a bola branca, o número de bolas pretas e, para cada bola preta, a sua posição inicial (linha e coluna) e o sentido de movimento, determine se a bola branca chega ao fundo do tabuleiro sem colidir com nenhuma das bolas pretas em movimento. Em caso de colisão, o programa deve imprimir Colide e a linha correspondente à bola preta que colide com a bola branca. Caso contrário o programa deverá imprimir simplemente que a bola Passa.


Input

Os dados do problema são apresentados da seguinte forma:
  • uma linha com dois inteiros L e C (<=50) que correspondem ao tamanho do tabuleiro;

  • uma linha com um só inteiro ColBranca que indica a coluna onde inicialmente se encontra a bola branca;

  • uma linha com um só inteiro NPretas que indica quantas bolas pretas vão ser usadas;

  • NPretas linhas, cada uma com três inteiros representando respectivamente: a linha onde a bola preta vai ser colocada, a coluna em que se encontra inicialmente e o sentido -1 (para a esquerda) ou 1 (para a direita) do movimento.


Output

O programa deverá apresentar como resultado:
  • passa: se não há colisão.
  • colide n: se há colisão, em que n é o número da linha em que se encontra a bola preta que colidiu com a bola branca:

Exemplo 1: input 


5 7
3
3
2 2 -1
5 1 1
4 6 1


Exemplo 1: output 


passa


Exemplo 2: input 


3 3
1
2
2 2 1
3 3 1


Exemplo 2: output 


colide 3


Exemplo 3: input 


4 2
2
3
4 2 -1
2 2  1
3 1  1


Exemplo 3: output 


passa
