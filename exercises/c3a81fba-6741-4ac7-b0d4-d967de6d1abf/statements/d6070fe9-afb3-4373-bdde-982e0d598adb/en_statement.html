AND, otherwise?

This year the national finals of the Olympic Games are held in Abobadeira. Numerology, whose main event is the Game of the Rest. We will support with the programming of the results table and we need your collaboration. You still don't know the Game of Rest? It's very simple:

  1. it is played with \ (N \) players, from player \ (0 \) to \ (N-1 \);
  2. each game has \ (M \) plays;
  3. in each move, each player registers an integer from \ (0 \) to \ (N-1 \);
  4. the numbers registered by all players are added, \ (S \);
  5. the remainder \ (R \) of the entire division of \ (S \) by \ (N \) is calculated, with \ (0 \ leq R \ leq N-1 \);
  6. the player with the number \ (R \) counts 1 more point, the the rest keep those who have;
  7. at the end of the \ (M \) moves, the winner is the one with the most points.

Task

Write a program to read the various moves, count the points of the players and indicate the winners.

Input

The first line has two integers: \ (N \) (the number of players) and \ (M \) (the number of moves). Following are \ (M \) lines containing the plays. Each line with a move contains \ (N \) integers \ (J_0, J_1, \ ldots J_i, \ ldots J_ {N-1} \), with \ (0 \ leq i

Restrictions

\ (1 \ leq N <100 \) Number of players
\ (1 \ leq M <100 \) Number of moves

Output

The result consists of 2 groups of lines, each line with only one whole. The lines in the first group have the total points for each player, in ascending order of the player's number. Between the first and the second group of lines there is a blank line. The lines of the second group show the numbers of the players who totaled the highest number of points and for that reason they won the game, also in ascending order of their number.

Example 1

Input 

2 4
0 0
0 1
1 0
1 1

Output 

two
two

0
1

Example 2

Input 

3 5
0 1 2
2 1 0
0 0 0
1 1 2
0 2 0

Output 

3
1
1
 
0
