O D. João Segundos clock

Problem

D. João Segundos, ingenious Portuguese inventor, created a watch based on a single time indicator: a seconds counter. This watch showed on the form of a negative integer the seconds that were left to noon, or as a positive integer or null the seconds that they passed since the same noon. In this simple way, D. João Segundos he always knew what time it was.

Task

Write a program that converts the seconds counter to the usual representation of digital clocks (HH: MM: SS).

Input

The input is the integer in the range of [-43200,43199] that represents the seconds counter.

Output

The program should write as a result the digital representation of the indicated time. To do this, you must use two digits to represent each of the three components: hours, minutes and seconds, separating the components by the : symbol.

Example 1: Input 



0

Example 1: Output 



12:00:00

Example 2: Input 



-32122

Example 2: Output 



03:04:38

Example 3: Input 



4571

Example 3: Output 



13:16:11
