Problem E: O Relógio de D. João Segundos

Problema

D. João Segundos, engenhoso inventor português, criou um relógio baseado num único indicador de tempo: um contador de segundos. Este relógio mostrava sobre a forma de um inteiro negativo os segundos que faltavam para o meio-dia, ou sobre a forma de um inteiro positivo ou nulo os segundos que passavam desde o mesmo meio-dia. Desta forma simples, D. João Segundos sabia sempre que horas eram.

Tarefa

Escrever um programa que converta o contador de segundos para a representação usual dos relógios digitais (HH:MM:SS).

Input

O input é o inteiro no intervalo de [-43200,43199] que representa o contador de segundos.

Output

O programa deve escrever como resultado a representação digital do tempo indicado. Para isso deverá utilizar dois dígitos para representar cada uma das três componentes: horas, minutos e segundos, separando as componentes pelo símbolo :.

Exemplo 1: Input 



0

Exemplo 1: Output 



12:00:00

Exemplo 2: Input 



-32122

Exemplo 2: Output 



03:04:38

Exemplo 3: Input 



4571

Exemplo 3: Output 



13:16:11
