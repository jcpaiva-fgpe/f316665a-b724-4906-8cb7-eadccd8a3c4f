Perimeter of a Rectangle

Problem

A rectangle is a quadrilateral polygon in which all its angles are straight. The perimeter of a rectangle can be calculated by adding the length of its sides.

Task

Make a program that given the size of two adjacent sides of a rectangle calculate its perimeter.

Input

The input is given as two integers (less than ) that represent the length (in meters) of two adjacent sides of the rectangle.

Output

The output will be an integer that corresponds to the perimeter value of the rectangle (in meters).

Example 1: input 


10 5

Example 1: output 


30

Example 2: input 


5 5

Example 2: output 


20
