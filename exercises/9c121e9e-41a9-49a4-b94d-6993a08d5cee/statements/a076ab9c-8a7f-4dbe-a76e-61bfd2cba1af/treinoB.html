Problema B: Falta de Pontaria

Problema

Uma das atracções duma feira popular é o lançamento de bolas de trapos a objectos estrategicamente colocados para dificilmente poderem ser derrubados. Um grupo de amigos adquiriu N bolas de forma que cada um deles efectuasse exactamente J ou J+1 lançamentos. Cada bola é lançada só uma vez. Quantos elementos tem no máximo o grupo? Quantos efectuam  J lançamentos e quantos efectuam J+1?

Tarefa

Escrever um programa que, dada uma sequência de pares (N, J) terminada por 0 0, cada um dos quais correspondendo a um cenário, responda às três questões anteriores. Sabe-se que N é maior ou igual que J e não excede 10000.

Exemplo input 


8 3
10 3
0 0

Exemplo output 


2 0 2 
3 2 1
