Lack of aim

Problem

One of the attractions of a popular fair is the launch of balls of rags to objects strategically placed so that they can hardly be overturned. A group of friends acquired N balls so that each perform exactly J or J + 1 launches. Each ball is launched only once. How many elements does the group have at most? How many make J launches and how many make J + 1 ?

Task

Write a program that, given a sequence of pairs ( N , J ) ending in 0 0, each of which corresponds to a scenario, answer the previous three questions. N is known to be greater than or equal that J and does not exceed 10000.

Example input 


8 3
10 3
0 0

Example output 


2 0 2
3 2 1
