Music, Maestro!

Problem

In Western music, we find seven musical notes, ordered by increasing heights:

OF RE MI FAN SUN THERE SI OF RE MI ...
1 two 3 4 5 6 7 1 two 3 ...

By definition, a musical interval is the difference that exists between two grades. As we have seven musical notes, and taking into account that the same note can form an octave range with itself, we also find seven musical intervals.

Depending on the number of degrees that exist between the first and the second note of the interval, it can be classified as second, third, fourth, fifth, sixth, seventh and finally eighth. Consider, for example, the notes from there to C. Following the sequence of degrees, there, if and do, we count three degrees and conclude that it is an interval of third.

Task

Write a program that gives two musical notes, identified by respective number, calculate the number of degrees between them and determine the musical break.

Input

The first line of an input is the first musical note, the second line of an input is the second musical note.

Output

The output will be a line that shows the name of the range.

Example 1: Input

2
4

Example 1: Output

third interval

Example 2: Input

4
2

Example 2: Output

sixth break

Example 3: Input

1
7

Example 3: Output

seventh interval
