Calligraphy

We know that in order to learn to write with reasonable handwriting, it is it is necessary to repeat the same letter several times. That's what the program is going to simulate.

Task

Write a program to repeat each letter several times indicated, being the number of times you will repeat defined for each one.

Input

A line with the number of cases to be treated, followed by a sequence of lines, each presenting a case. Each case is defined by a letter uppercase followed by a digit (1, ..., 9), which represents the number of times that you will have to repeat that letter.

Output

Each line has a sequence of equal letters, which corresponds to the response to the calligraphy training exercise.

Example

Input 

two
T1
S9

Output 

T
SSSSSSSSS
