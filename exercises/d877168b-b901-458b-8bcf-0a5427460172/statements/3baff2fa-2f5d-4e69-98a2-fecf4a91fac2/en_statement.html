Mastermind

Mastermind is a game for two people who play the role of codemaker and codebreaker :

  • the codemaker chooses a secret key (a sequence of four pins chosen from six colors);
  • the codebreaker will deduce which key is using the smallest number of attempts.
For each attempt the codemaker says two scores: the number correct colors in the correct position ; and the number of colors correct but in the wrong position .

We represent the six colors by the lower case letters from ' to ' a ' f '; if the key is, for example, " acfb " and the attempt " abcd " the scores are (1,2), that is, a letter (' a ') correct in the correct position and two (' c ', ' b ') correct but in the wrong positions.

Repeated colors may occur. For example, if the key is " acfa " and the attempt " aaac " the scores are (1,1), or that is, an ' a ' in the correct position and an ' a ' in position wrong (the third ' a ' in the attempt does not count).

Task

We want a program that reads the key and the attempt and calculates the two scores.

Example 1

Input 

acfb
a B C D

Output 

1
two

Example 2

Input 

acfa
aaac

Output 

1
two
