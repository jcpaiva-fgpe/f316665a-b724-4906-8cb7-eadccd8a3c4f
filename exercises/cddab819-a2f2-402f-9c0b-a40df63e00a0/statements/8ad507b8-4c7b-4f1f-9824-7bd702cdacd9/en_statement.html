Digital Clock

Problem

The most common digital clocks use the so-called seven dials segments, where each digit is constructed from three segments horizontal and four vertical, which can be lit or off. O mapping digits in lit segments to the digits 0 to 9 is the Following:

Task

Write a program that, starting from the time in HH: MM format (clock 24 hours) print the corresponding image of the seven segment display.

Input

The time in HH: MM format (24-hour clock).

Output

The image of the seven-segment dial, built from the characters `` - '', `` | '' and space, with the layout Following:

Note that between the two digits of the hours, as well as between the two minute digits, there is a vertical column of spaces; between the hours and minutes there is a `` : '' sign; all lines of output is 15 characters long.

Example: input 


12:35

Example: output 


     - - -
  | | | |
     -: - -
  | | | |
     - - -
