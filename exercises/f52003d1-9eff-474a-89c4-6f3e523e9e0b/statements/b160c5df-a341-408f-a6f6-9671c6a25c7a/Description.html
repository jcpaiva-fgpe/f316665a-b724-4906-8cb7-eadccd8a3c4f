Jogo do Galo

Problema

O jogo do galo joga-se com dois jogadores num tabuleiro de dimensão 3*3. Cada jogador coloca alternadamente uma das suas peças numa das posições por preencher. Ganha o jogador que conseguir colocar 3 peças suas em linha na horizontal, vertical ou diagonal. O jogo termina assim que um dos jogadores vença ou quando todas as posições estiverem ocupadas, situação esta considerada de empate.

Tarefa

Escrever um programa que, dado um tabuleiro com o estado final de um jogo, verifica se o estado é correcto ou não. Se o estado for correcto, o programa deve ainda determinar se existe algum vencedor ou se o jogo terminou empatado. As posições no tabuleiro são representadas pelos valores 0, 1 e 2. O valor 0 representa uma posição por preencher, o valor 1 representa uma peça do jogador 1, e o valor 2 representa uma peça do jogador 2. O jogador 1 é sempre o primeiro a jogar.

Input

O input é o estado final do tabuleiro. O tabuleiro é dado por três linhas, cada uma com três inteiros (0, 1 ou 2) representado as peças colocadas nas posições dessa linha.

Output

O programa deverá apresentar como resultado:

jogo incorrecto: se o estado do jogo não é correcto.

venceu jogador 1: se o estado do jogo é correcto e se venceu o jogador 1.

venceu jogador 2: se o estado do jogo é correcto e se venceu o jogador 2.

empate: se o estado do jogo é correcto e se terminou com um empate.

Exemplo 1: input 


1 2 1
2 2 1
1 0 2

Exemplo 1: output 


jogo incorrecto

Exemplo 2: input 


1 0 2
1 0 2
1 0 2

Exemplo 2: output 


jogo incorrecto

Exemplo 3: input 


1 1 2
0 2 0
2 0 1

Exemplo 3: output 


venceu jogador 2
