Game of the Rooster

Problem

The cock game is played with two players on a 3 * 3 board. Each player alternately places one of his pieces in one of the positions unfilled. The player who manages to put 3 of his pieces in wins line horizontally, vertically or diagonally. The game ends as soon as a of the players win or when all positions are occupied, situation is considered a tie.

Task

Write a program that, given a board with the final state of a check whether the status is correct or not. If the status is correct, the The program must also determine if there is a winner or if the game ended in a draw. The positions on the board are represented by values ​​0, 1 and 2. The value 0 represents an unfilled position, the value 1 represents a piece by player 1 , and the value 2 represents a piece from player 2 . Player 1 is always the first to play.

Input

The input is the final state of the board. The board is given by three lines, each with three integers (0, 1 or 2) representing the pieces placed in the positions of that line.

Output

The program should present as a result:

incorrect game : if the state of the game is not correct.

beat player 1 : if the state of the game is correct and if won player 1.

beat player 2 : if the state of the game is correct and if won player 2.

empate : if the state of the game is correct and if it ended with a tie.

Example 1: input 


1 2 1
2 2 1
1 0 2

Example 1: output 


incorrect game

Example 2: input 


1 0 2
1 0 2
1 0 2

Example 2: output 


incorrect game

Example 3: input 


1 1 2
0 2 0
2 0 1

Example 3: output 


won player 2
