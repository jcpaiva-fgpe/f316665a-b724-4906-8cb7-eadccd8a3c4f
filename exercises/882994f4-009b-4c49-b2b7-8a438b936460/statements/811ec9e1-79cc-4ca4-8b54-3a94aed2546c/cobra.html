Problema E
Labirinto da cobra

Problema

Uma cobra move-se num labirinto onde existem vários obstáculos e vários ovos. O objectivo da cobra é, partindo de uma posição inicial pré-definida, comer todos os ovos existentes no labirinto (i.e. passar por todos os pontos em que existem ovos) sem chocar contra os obstáculos nem sair do labirinto. Durante o jogo a cobra pode mover-se para a direita, esquerda, para cima e para baixo (sem nunca sair do labirinto).

Tarefa

Escrever um programa que, dados um labirinto e vários caminhos devolva se os caminhos são ou não válidos, isto é, se a cobra consegue para cada um dos caminhos comer todos os ovos sem sair do labirinto e sem chocar contra os obstáculos.

O labirinto consiste numa matriz quadrada cuja dimensão é dada. Os obstáculos estão assinalados no labirinto como #. Os ovos estão assinalados como O. A cobra ocupa apenas uma posição da matriz e está assinalada como C.

Input

As duas primeiras linhas de input são dois inteiros inferiores a 100,

n - o tamanho do labirinto;
m - o número de caminhos a testar.

As n linhas seguintes, cada uma com n caracteres, descrevem o labirinto e são constituidas apenas pelos seguintes caracteres:

O - representa um ovo;
C - representa a posição inicial da cobra;
# - representa um obstáculo;
. - representa um espaço livre.

As m linhas seguintes descrevem um caminho com tamanho menor do que 100 e são constituidas apenas pelos seguintes caracteres:

e - movimento para a esquerda;
d - movimento para a direita;
c - movimento para cima;
b - movimento para baixo;

Output

O seu programa deve escrever SIM ou NAO conforme o caminho é ou não válido para o labirinto dado.

Exemplo de Input 

6
4
######
#C..O#
#.#..#
#.#..#
#O#..#
######
dddeeebbb
ddeebbb
ddbbdcceeebbb
ddddeeebbb

Exemplo de Output 

SIM
NAO
SIM
NAO
