Learning to write

To learn to write with reasonable handwriting, it is necessary to repeat the same letter several times. This is what the program will simulate.

Task

Write a program to repeat each letter several times indicated, being the number of times you will repeat defined for each one.

Input

A line with the number of cases to be treated, followed by a sequence of lines each presenting a case. Each case is defined by a letter uppercase followed by a positive integer less than 50, which represents the number of times you will have to repeat that letter.

Output

Each line has a sequence of equal letters, which corresponds to the response to the calligraphy training exercise.

Example

Input 


5
T13
O1
P7
A5
S10

Output 


TTTTTTTTTTTTT
O
PPPPPPP
YYYYY
SSSSSSSSSS
