Problem B: Aprender a escrever

Para aprender a escrever com uma caligrafia razoável, é necessário repetir várias vezes a mesma letra. É isso que o programa vai simular.

Tarefa

Escrever um programa para repetir várias vezes cada uma das letras indicadas, sendo o número de vezes que vai repetir definido para cada uma.

Input

Uma linha com o número de casos a tratar, seguida de uma sequência de linhas cada uma apresentando um caso. Cada caso é definido por uma letra maiúscula seguido de um inteiro positivo menor do que 50, que representa o número de vezes que terá de repetir tal letra.

Output

Em cada linha tem uma sequência de letras iguais, que corresponde à resposta ao exercício de treino de caligrafia.

Exemplo

Input 


5
T13
O1
P7
A5
S10

Output 


TTTTTTTTTTTTT
O
PPPPPPP
AAAAA
SSSSSSSSSS
