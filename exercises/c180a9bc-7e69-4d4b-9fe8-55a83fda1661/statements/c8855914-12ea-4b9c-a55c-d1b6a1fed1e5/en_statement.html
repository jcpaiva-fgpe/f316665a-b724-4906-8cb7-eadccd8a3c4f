Survival Manual

How to spend time on a long journey, if you can't read it, don't can I use the phone, the mp3, the game console, ..., or anything? Perhaps it only remains to count signposts or to appreciate the landscape, or to sleep! The plates are then counted.

Task

Write a program to count the number p and P 's in a sequence of letters ( p and P represent plaques). THE sequence is given on a line, which ends with a line change and has at least least one letter. You can assume that it will be no longer than 1000 characters if useful to you. The program will write the number of plates followed by a change of line.

Example

Input 


TOPASDCCFCUPPPpptopaspucfSOPATDcc

Output 


9
