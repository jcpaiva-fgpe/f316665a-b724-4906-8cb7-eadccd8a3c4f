Art or ingenuity?

- Mario, I can guess numbers by telepathy. - said Marta. - Think of a number and don't tell me.

Mário nodded, and replied:

- I've thought.

- If the number you thought is even, multiply it by five; otherwise, it multiplies for seven - continued Marta. - Now, add up to five; multiply by two; adds up to 10; divide by two, is it already? Add five again and multiplies by three. You haven't gotten lost yet, have you? Well, if the number in that you had thought was unique, now add the quadruple of that number. Already it’s almost there, you just need to divide it by five Now, I think you already thought enough in the number for me to guess it. How much did I give you?

- Thirty three. - Mário replies, curious.

- Aah! Ah! - exclaims Marta, victorious but tired of so many accounts of head - So you thought of number eight, right?

- It was was. - Mário nodded, very puzzled. And he said to his buttons: `` Is this art or ingenuity? Telepathy or trick? ''

Task

Make a program to help Marta guess the number that Mário thought. The program must determine the number that Mário thought.

Input

A positive integer that represents Mário's answer.

Output

A line with an integer that represents Marta's answer, guessing the number that Mário initially thought of. If it does not exist an integer that can give rise to the number that Mário obtained, the Marta will reply, `` I think you were wrong in the accounts. '' (Without quotes).

Example 1

Input 

27

Output 

6

Example 2

Input 

24

Output 

3

Example 3

Input 

30

Output 

I think you made a mistake on the bills.
