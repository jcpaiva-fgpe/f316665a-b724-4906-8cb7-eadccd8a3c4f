Skating

There are sports, such as figure skating, in which there is no value easily observable and measurable to gauge athletes' merit. No there are balls entering the goal, running times or bar heights. THE evaluation is made by a jury and each element assigns a classification individual. The final classification is the average of the classifications of elements.

Even so, sometimes there are elements that give very high ratings different from the rest, which can significantly distort the final classification. So it is customary to withdraw the extreme values before averaging. However, if extreme values ​​occur more than once, that is, if more than one member of the jury assigns this classification (maximum or minimum), the corresponding value is taken significant and is considered for the calculation of the average.

Task

It is intended a program that, given the n classifications assigned by the n members of the jury, calculate the final classification. These classifications are integers between 0 and 100.

The final classification must be calculated as an integer value. That is, the entire division must be used to divide the sum of the values considered by the number of values ​​considered.

The first line of the data contains the value of n , which is always greater than 2 and less than or equal to 10. Next are n non-integers negative and less than or equal to 100, which are the individual assignments.

The result consists of a single line with a non-negative integer and less than or equal to 100.

Example

Input 

5
45
55
43
55
44

Output 

49
