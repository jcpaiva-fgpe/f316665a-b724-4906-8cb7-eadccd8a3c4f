Sequência de Dígitos

Problema

Numa prova com 10 jogadores (numerados de 0 a 9), ganha o concorrente com o maior número de respostas correctas consecutivas. A prova desenrola-se da seguinte forma: um concorrente é seleccionado aleatoriamente para responder. Sempre que der uma resposta certa o concorrente ganha um ponto e mantém o direito de resposta. Sempre que der uma resposta errada o concorrente perde o direito a responder sendo aleatoriamente seleccionado outro concorrente. A informação de uma prova consiste numa sequência de dígitos que correspondem aos números dos jogadores que responderam acertadamente às respostas. Por exemplo na prova 
0 0 2 3 5 5 4 4 4 7 5 5 5 5 1 9
O vencedor foi o concorrente 5 com um máximo de 4 respostas certas consecutivas.

No caso de empate, ganha o primeiro jogador a obter a sequência máxima de respostas certas. Por exemplo na prova 

0 2 3 4 4 4 7 5 5 1 1 1 9 4 4 5 5 5
O vencedor foi o concorrente 4 pois foi o primeiro a obter 3 respostas certas consecutivas.

Tarefa

Escrever um programa que, dada uma sequência de dígitos (0, 1, ..., 9), contendo a informação de uma prova, determine qual o vencedor da prova e o número de respostas que lhe deu a vitória. Ou seja, determine qual o tamanho da maior sub-sequência de ocorrências consecutivas do mesmo dígito (número de respostas) e para que dígito ocorreu esse máximo (número do jogador).


Input

A primeira linha do input é um inteiro N (<=1000) que indica o tamanho da sequência a considerar. A segunda linha é a sequência de N dígitos que representa a informação da prova.

Output

O output é uma linha com os dois inteiros, representando o número do jogador que ganhou a prova e o número de respostas certas consecutivas que lhe deu a vitória.

Exemplo de Input 


20
0 0 2 3 3 3 4 5 5 6 3 3 3 3 3 2 3 3 3 3

Exemplo de Output 


3 5
