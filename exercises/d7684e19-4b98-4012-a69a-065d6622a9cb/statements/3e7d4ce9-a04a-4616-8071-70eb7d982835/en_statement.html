Good Luck South Africa

Portuguese sportsmanship is well known. So, intending to accompany the Selection to the World Cup in South Africa, some Portuguese have learned to say "good luck" in several languages: 

        GOOD LUCK

        BUENA SUERTE

        BONNE CHANCE

        BUONA FORTUNA

        VIEL GLUECK

Assignment

Write a program that reads two integers n and l and write n times good luck in the l language (where l represents the order number in the previous list).

Input

In the first line there is the positive integer . On the line next has the value of l (where ).

Output

There will be n lines, in exactly the same words.

Example 1

Input 

4
two

Output 

BUENA SUERTE
BUENA SUERTE
BUENA SUERTE
BUENA SUERTE

Example 2

Input 

1
4

Output 

BUONA FORTUNA
