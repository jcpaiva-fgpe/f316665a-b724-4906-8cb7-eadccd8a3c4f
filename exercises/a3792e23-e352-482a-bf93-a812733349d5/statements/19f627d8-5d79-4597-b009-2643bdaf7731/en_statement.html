Boas-Fitas Ticket Office

The cinema screens Boas-Fitas, Lda need a program to ticket booking management and are recruiting programmers for the develop. The movie theaters are rectangular. Each place is identified by a row and column. Queues are identified by letters consecutive capital letters (A to Z maximum). The columns are identified consecutive positive integers (1 to 99 maximum). There are two types of reservation requests: exact reservations , in which the queue and the start column and the number of contiguous places desired, and free reservations , where the number of contiguous seats is given intended, these must be completed on a first-come, first-served basis. starting in row A, and in ascending order of columns. The places reserved for an order are in a single row.

Task

Write a program to process a list of reservation requests for tickets to a screening in a movie theater and answer every request according to previous reservations.

Input

The entry starts with two positive integers that represent the number of rows and columns in the room. Next comes the number of reservation requests, followed by the orders themselves, one on each line. Requests for exact reservation starts with X followed by a line and column, a space and the number of seats. Free booking requests start with L followed by the number of seats.

Output

In each line you have the answer to a reservation request, with the responses presented in order of arrival. Answers can be OK followed by a space, the starting row and column and the number of reserved seats or NA if the reservation cannot be made.

Example

Input 

3
10
5
L3
XA4 3
L4
XA1 2
L2

Output 

OK A1 3
OK A4 3
OK A7 4
AT
OK B1 2
