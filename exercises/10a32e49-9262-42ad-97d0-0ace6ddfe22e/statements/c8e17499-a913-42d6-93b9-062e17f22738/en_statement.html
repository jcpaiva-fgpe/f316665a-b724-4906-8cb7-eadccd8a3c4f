Sick of the ball

As the end of the championship approaches, the sick of the ball cling to the calculator to find out if they can already party or work on the rationalization. The accounts are not even difficult to do. In each of the missing matches are 3 points up for grabs. If the nearest opponent get all those points, our team none, and even then we have more points, so the championship is already ours. If it's the other way around, we have say goodbye to the championship. But it can also happen that if we lose points and they win either team can still win.

Task

Given the points of our team, the closest opponent, and the number of missing days (there are 3 points in dispute in each of them), determine which case our team is in, identified by a smiley , namely:

:-D we are champions - even if we lose all the games and they win all, you won't be able to get more points than us;
:-) we are the biggest - we have more points than the opponent and they they only win if we lose too many points;
: - | there is still hope - we have less points than the opponent but we can still be winners;
:-( for the year it's our turn - even if we win every game and they lose everyone we no longer have a chance.

To simplify the accounts we will assume that the team at this moment with more points have a much greater number of goals (unbeatable). Therefore, if the number of points at the end of the championship is equal, it will be the champion team that currently has the most points. Let's also assume that in this teams currently have different scores.

Input

There is only one line of input with 3 integers, in this order:

  • the number of points of our team
  • the number of points from our closest opponent
  • the number of matches to be played

Output

Only one line of output with an appropriate smiley , depending on the point difference and the explanation above.


Example 1

Input 

30 20 3

Output 

:-D

Example 2

Input 

30 21 3

Output 

:-D

Example 3

Input 

30 22 3

Output 

:-)

Example 4

Input 

22 30 3

Output 

: - |

Example 5

Input 

21 30 3

Output 

:-(

Example 6

Input 

20 30 3

Output 

:-(
