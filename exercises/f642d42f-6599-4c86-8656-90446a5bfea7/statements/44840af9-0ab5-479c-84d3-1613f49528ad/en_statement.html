Anagrams

Two words or phrases are anagrams if they are written with the same letters used the same number of times but eventually in positions many different. For example: `` optas '' and `` ToPAS '' are anagrams as well as `` Clint Eastwood '' and `` Old West Action '' (ignoring spaces and distinction between uppercase and lowercase).

Task

Write a program that checks if two texts are anagrams. Each one will be no longer than 200 characters. You should consider equivalent to uppercase and lowercase letters and ignore all non-letter characters (spaces, punctuation marks, etc.); can assume that the letters have no accents. The entry consists of two lines of text, one per sentence. The exit should be yes if the two sentences are anagrams and no contrary.

Example 1


Input 


Clint eastwood
Old West Action

Output 


yes

Example 2


Input 


ToPAS-SUL
ToPAS

Output 


at the
