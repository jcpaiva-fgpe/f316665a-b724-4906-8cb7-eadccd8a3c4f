What is the greatest number of equal pieces that can be obtained? Like there are several tapes of the same type to analyze, some wider than others, some longer than others, we will try to automate the process.

Task

Write a program that, given the description of the tape to be cut, indicates the maximum number of equal pieces that can be obtained by applying cuts transversal (in the direction of the width, that is, perpendicular to the line of shot).

Input

In the first line two integers L and C representing the tape width and length respectively. The width and the length correspond to the number of rows of symbols. You can assume that the L never exceeds 8 and C does not exceed 10000. On lines The following is the description of each line of the tape (depending on the width). O T character exists only at the end of each line and will not count towards the tape width.

Output

You will write an integer that indicates the number of pieces you would get (followed by line break).
